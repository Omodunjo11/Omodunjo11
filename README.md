## Selected Work

### Enterprise AI & Identity Infrastructure
Designed and shipped OAuth 2.0 / OIDC implementation via Auth0 across web and admin tooling, including RBAC architecture, JWT lifecycle management, step-up MFA triggers, and system-of-record enforcement.

Architected dual-memory RAG systems with grounding-first generation, source isolation, abstention policies, and evaluation harnesses tracking precision/recall, drift, latency (P50/P95), and token sensitivity.

### Enterprise LLM Classification Platform
Reduced cost-to-serve 62% and improved first-pass resolution 41% across 12 enterprise clients by replacing deterministic rule trees with probabilistic LLM classification engines.

Built evaluation pipelines, production FastAPI modules, pgvector retrieval layers, and inference monitoring inside SOC 2-equivalent environments.

### C++ Vector Search Engine
Built multithreaded C++ vector search engine (100K × 768-dim embeddings), reducing latency ~37% through parallelization and memory-aware optimization on ARM architecture.

Benchmarked scalar vs SIMD (NEON) implementations; diagnosed memory bandwidth saturation as dominant bottleneck.
